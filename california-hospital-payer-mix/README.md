# California Hospital Inpatient Payer Mix Analysis

Live Dashboard: https://public.tableau.com/app/profile/michael.lee6058/viz/CaliforniaHospitalInpatientPayerMix/CaliforniaHospitalInpatientPayerMixDashboard

SQL-based analysis of how hospital inpatient care is paid for across California counties, using HCAI public patient discharge data from 2012 to 2024.

## Key Questions
1. What is the statewide payer mix across all California counties?
2. Which counties have the highest Medi-Cal utilization?
3. Which counties have the highest uninsured Self-Pay burden?
4. How does payer mix compare across the Central Valley, Bay Area, and Southern California?
5. How has statewide payer mix shifted between 2012 and 2024?

## Key Findings
- Medi-Cal is the largest payer statewide, reflecting California's large low-income population
- Central Valley counties consistently show higher Medi-Cal rates than Bay Area counties
- Self-Pay rates are highest in counties with large uninsured populations
- Private health insurance share has shifted over the 2012 to 2024 period

## Tools Used
Python, SQL (pandasql), Pandas, Matplotlib

## Data Source
HCAI Hospital Inpatient Characteristics by Patient County of Residence
https://data.chhs.ca.gov/dataset/hospital-inpatient-characteristics-by-patient-county-of-residence
